## Task 99

**Score:** 4

### Input
```
次の物語の展開を予想してみましょう。

ある日、主人公のもとに不思議な手紙が届きました。手紙には「今夜の満月に、森の奥深くにある洞窟に来てください。あなたを待っています。」と書かれていました。主人公はその手紙に従い、夜中に洞窟の入り口にたどり着きました。中に入ると、謎めいた人物と出会い……。
```

### Model Output
```
面白そうな物語の始まりですね。展開を予想してみます。

謎めいた人物は、主人公を迎え入れると、洞窟の奥深くに案内しました。主人公は、不思議な手紙に導かれてこの場所に来てしまったが、ここが一体何の用途で使われているのか全く見当がつきません。

謎めいた人物は、主人公に近づき、口を開きました。「あなたは、選ばれし者です。私が待っていたのは、実はあなたではありません。手紙に書かれていた内容は、誤りでした。」
主人公は、混乱します。何のことかわからないが、選ばれし者と呼ばれ、手紙の内容が誤りだったと知らされます。

謎めいた人物は、主人公に、真相を話すと、実は洞窟の奥深くに、ある秘密が隠されていました。主人公は、驚くべき真実を知らされます。

秘密とは、主人公の過去にまつわることや、家族、友人、職業など、全てが偽りであったと知らされます。主人公は、自分のアイデンティティが崩壊します。

主人公は、混乱に陥り、逃げるように洞窟から出て、手紙を書いた人物を捜索します。捜索の結果、手紙を書いた人物は、実は主人公の未来の自分であることがわかります。

未来の自分は、主人公に、過去
```
